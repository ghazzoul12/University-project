Accuracy function

acc_error <- function(actual, pred) {
  mape <- mean(abs((actual - pred)/actual)) * 100
  mae <- mean(abs(actual - pred))
  RMSE <- sqrt(mean((actual - pred)^2))
  vec <- c(mape, mae, RMSE)
  names(vec) <- c("MAPE", "MAE", "RMSE")
  return(vec)
}

Lasso Regression

Lasso regression, or Least Absolute Shrinkage and Selection Operator, is a type of linear regression that performs both variable selection and regularization to enhance the prediction accuracy and interpretability of the statistical model it produces. The key feature of Lasso regression is that it can shrink some of the coefficients to zero, effectively selecting a simpler model that retains only the most impactful variables.

Lambda () is the tuning parameter in Lasso regression that controls the strength of the penalty applied to the coefficients. A higher value of lambda results in more regularization, shrinking coefficients more aggressively, whereas a lower value applies less regularization.

Choosing the optimal lambda is crucial for achieving the best predictive performance. This is typically done using cross-validation, a technique that involves dividing the data into multiple subsets, training the model on some subsets, and validating it on the remaining subsets. By evaluating the models performance across different values of lambda, we can select the lambda that minimizes the cross-validation error. The use of cross-validation for selecting the optimal lambda ensures that the model is robust and less sensitive to variations in the training data. This leads to more reliable and stable predictions.
